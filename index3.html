jQuery 애니메이션 관련 메서드(함수)

  1. $('대상').fadeIn() -> 점점 나타나기
  2. $('대상').fadeOut() -> 점점 사라지기
  3. $('대상').fadeToggle() -> 점점 나타나기/사라지기
  4. $('대상').slideUp() -> 요소의 높이를 0으로 애니메이션
  5. $('대상').slideDown() -> 요소의 높이를 설정한 높이대로 에니메이션
  6. $('대상').slideToggle() -> 한번 누르면 요소의 높이를 0, 다시 누르면 설정한 높이대로
  7. $('대상').animate() -> 요소를 이동시킬때
